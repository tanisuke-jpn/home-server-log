# Pi-hole


## 概要

家庭内ネットワーク向けDNS広告ブロックサーバ。


## 目的

不要な広告やトラッキング通信を
ネットワーク単位で制御する。


## 構成

```
端末

 ↓

Pi-hole(DNS)

 ↓

Internet
```


## Docker

コンテナとして運用予定。


## 利用方法

ルーターまたは端末のDNS設定を
Pi-holeへ変更する。


例：

```
DNS:
192.168.0.8
```


## 注意

DNS設定を変更すると、
家庭内ネットワーク全体へ影響する。


## メモ

問題発生時に切り戻せるよう、
既存DNS設定を記録しておく。
