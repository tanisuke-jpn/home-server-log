# ThinkStation P330 Tiny


## 概要

自宅サーバのメイン機として使用。

小型筐体ながら、
常時稼働用途に適した性能を持つ。


## ハードウェア

|項目|内容|
|-|-|
|メーカー|Lenovo|
|機種|ThinkStation P330 Tiny|
|CPU|Intel Core i7-8700T|
|Memory|32GB|
|GPU|NVIDIA Quadro P620|
|Storage|NVMe SSD|
|OS|Ubuntu Server 24.04|


## 用途

予定しているサービス：

- ファイルサーバ
- 音楽サーバ
- 動画配信
- 同期サーバ
- DNS広告ブロック
- 録画管理


## BIOS設定

確認済み項目：

|項目|設定|
|-|-|
|Boot Mode|UEFI Only|
|Virtualization Technology|Enabled|
|VT-d|Enabled|


## メモ

省電力CPUのため、
24時間稼働を想定。
